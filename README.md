## 🎯 프로젝트 목적
### 무인 정밀 농약 분사 차량과 주행/살포 기록 시스템
+ 인구 감소로 인한 노동력 감소 및 정밀 농업 분야에 대한 솔루션.
+ 주행 경로와 제초제 살포 위치를 지도 데이터로 저장하여 농장 운영에 대한 방향성 제시.
